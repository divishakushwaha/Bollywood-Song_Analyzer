# Bollywood-Song_Analyzer
Built and deployed an AI-powered Bollywood song mood analyzer using Groq's LLaMA 3.3 70B model, with a Streamlit front-end hosted on Streamlit Cloud — analyzes mood, psychology, and emotion of any song in real time

An AI-powered web app that analyzes the mood and psychology of any Bollywood song.

🔗 **Live demo:** https://your-app-link.streamlit.app

## What it does
Type any Bollywood song name
Get instant AI analysis of the mood, emotions, and psychology and why people connect with the song
- Get 3 similar song recommendations according to its emotion

## Tech stack
Python · Streamlit · Groq API · LLaMA 3.3 70B

## Run locally
pip install -r requirements.txt
streamlit run bollywood.py
